Vocab Builder

In Vocab Builder, which is a tool to help you learn words, you can search for definitions of words, then when you are done learning them, you can enter them in the learning list which will delete it from the learning list and add it to the mastered list. If you are still unsure about the word, you can move the word back from the mastered list to the learning list.

Whatsapp Bot

I used Selenium to make Whatsapp Bot. What it does is basically you input a message and a phone number or multiple phone numbers. What it will then do is open whatsapp web and send it to them. I used inspect to find the XPATH of relevants elements like buttons so the bot would know where to click. This could be useful for things like sending party invites to many people at once.
